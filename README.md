# Cyber Attack Predictor

## Datasets de Prueba

- [Web Server Access Logs](https://www.kaggle.com/datasets/eliasdabbas/web-server-access-logs/data)
- [HTTP Dataset CSIC 2010 | Cookie Security & Vulnerabilities Dataset - CISC](https://www.kaggle.com/datasets/shashikiran42/cookie-security-and-vulnerabilities-dataset-cisc) - The HTTP dataset CSIC 2010 contains thousands of web requests automatically generated. It can be used for the testing of web attack protection systems. It was developed at the Information Security Institute of CSIC (Spanish Research National Council)
- [Web Attack Payloads | Web Application Payloads Dataset](https://www.kaggle.com/datasets/cyberprince/web-application-payloads-dataset) - A collection of web attack payloads.
